# Vivid Midnight Theme

A combination of [Material Theme](https://github.com/equinusocio/vsc-material-theme) and [Monokai++](https://github.com/dcasella/monokai-plusplus) syntax colors.  
This is a **work in progress** project. If you have any problems, please open an [issue](https://gitlab.com/mrGibi/vscode-vivid-midnight-theme/issues).

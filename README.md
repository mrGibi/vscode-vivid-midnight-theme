# Vivid Midnight Theme

A VSCode theme inspired by [Material Theme](https://github.com/equinusocio/vsc-material-theme) ([Mattia Astorino](https://github.com/equinusocio)) and [Monokai++](https://github.com/dcasella/monokai-plusplus) ([Davide Casella](https://github.com/dcasella)).

This is my first VSCode theme and a **work in progress** project. If you have any problems or suggestions please open an [issue](https://gitlab.com/mrGibi/vscode-vivid-midnight-theme/issues).
